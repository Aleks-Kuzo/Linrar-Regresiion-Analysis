# Linear-Regresiion-Analysis
Сначала я сгенерировал данные для задачи регрессии: 50 объектов, которые состоят из 1 признака, при этом он один полезный, и визуализировал это.
Далее, чтобы оценить качество модели, посчитал метрику MSE - среднюю квадратичную ошибку, и её градиент.
Потом я реализовал цикл градиентного спуска.
И проделал то же самое для многомерной линейной регрессии.
